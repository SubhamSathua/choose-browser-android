Privacy Policy

Effective date: March 2026  ·  Choose Browser

Main Application

No Data Collection

Choose Browser does not collect, transmit, or store any personal data, usage data, analytics, or identifiers.

Local Storage Only

The only information stored on your device is your selected preferences:

  • Application theme (Light / Dark / System default)
  • Preview Page settings (JavaScript toggle state, Ad-Blocker mode)

This data never leaves your device.

No Data Sharing

We do not share, sell, license, or disclose any data to third parties, advertisers, or analytics providers.

No Network Requests by the App

Choose Browser itself makes no network requests. All internet activity is initiated by you when launching a browser or using the Preview Page.

Permissions

The following Android permissions may be requested solely to enable functionality within the Preview Page WebView:

  • Camera — only if a website explicitly requests camera access.
  • Microphone — only if a website explicitly requests audio capture.
  • Location — only if a website explicitly requests geolocation.
  • Storage / Media — only if a website requests file access.

These permissions are never used by the application itself.

Preview Page (Private WebView)

Sandboxed Browsing

The Preview Page is an isolated, incognito-style WebView. Cookies, cache, history, and all site data are cleared automatically when the session ends.

No Data Collected by Us

Choose Browser does not monitor, log, or record any URLs visited, content viewed, or interactions made within the Preview Page.

Third-Party Websites

Websites you visit inside the Preview Page may collect data independently, subject to their own privacy policies. We have no control over and bear no responsibility for the data practices of third-party websites.

  • Third-party cookies are blocked by default.
  • The Ad-Blocker feature filters known tracking and advertising domains.
  • JavaScript is disabled by default.

No Persistent Data

The Preview Page disables all persistent storage mechanisms including DOM storage, databases, form autofill, and password saving.

SSL / Security

SSL errors result in the page being blocked. Mixed content (HTTP within HTTPS) is not permitted.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the application update notes.

Contact

If you have questions regarding this policy, please contact us through the feedback option in Settings.
